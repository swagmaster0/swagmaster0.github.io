Amelioration

Released 30 October 2024

A physics-based, action-adventure game where every object can be grabbed and thrown

You're some kind of biological creature, made to destroy the rejects that came before you- and to accompany a poor, lonely AI.
Available on Itch.io




Developer Commentary

This game was made for the NZQA Technology Scholarship in 2024, which it won.
Does this count as award winning?

The scholarship is a written report based on a year long project, detailing the development, ideation, and research behind it. It can be accessed below.

Scholarship Report

If you want to know specifics about the development processes and ideation, read the scholarship report.


This was one of my first game development experiences - there were a lot of mistakes made that I've learned from since. For instance, the game story I find to be a bit corny - and it also appears rushed in some areas. Some levels are frustrating to beat, and there are only 10 levels.

It also doesn't help that the game was a little under-baked - it was developed within a 6 month period (April 2024 -> October 2024).

In a way, I might continue development to fix these mistakes - I still have ideas for this game, particularly for more content and story revisions. Although, I have no idea if this will ever come to fruition.
