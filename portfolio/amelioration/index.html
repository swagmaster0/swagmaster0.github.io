Amelioration


A physics-based, action-adventure game where every object can be grabbed and thrown

You're some kind of biological creature, made to destroy the rejects that came before you- and to accompany a poor, lonely AI.


This game was made for the NZQA Digital Technology Scholarship in 2024, which it won. The scholarship is a written report based on a year long project, detailing the development, ideation, and research behind it. The report I submitted can be found in this PDF document


This was one of my first game development experiences - there were a lot of mistakes made that I've learned from since then. In a way, I might continue development to fix these mistakes,
